Overview

School project for Software Start-up course. The chat room software concentrates on music streaming and connects music enthusiasts with similar interests. Artists can also use TuneBox for promoting their own material.

Features

The users need to register and they are divided to free users with limited number of private chat rooms and premium users with higher limit, who can also create public rooms. The front page supports showing featured public rooms and advertisements for free users. TuneBox synchronizes the current music video between the users and also displays chat room suggestions based on the user's preferred tags.

Users can search for music videos asynchronoously (displays suggestions while typing) and the results are ordered as they would be in YouTube. While creating a chat room, users can add keywords as tags, upload and crop a picture for the room and write name and description.

Upcoming and Other Planned Features

  • Playlist support
  • Voting mechanism (like for positive feedback to the room's admin/vote to skip a song)
  • Streaming live with camera input
  • For the sake of the course's goals, we also designed some ideas for various fictional billing mechanisms, e.g. monthly subscription and buying booster packs to raise the room limits.

Technical Details

The project is implemented using MEAN stack, which consists of MongoDB (NoSQL database), Express.js (web application framework), Angular.js (JavaScript MVC framework) and Node.js (the execution environment). TuneBox uses YouTube API to search for videos and as the video player.
