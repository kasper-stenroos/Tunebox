Overview

School project for Software Start-up course. The chat room software concentrates on music streaming and connects music enthusiasts with similar interests.

Features

The users are divided to free users with limited number of private chat rooms and premium users with higher limit, who can also create public rooms. TuneBox is synchronizes the current music content between the users and also displays chat room suggestions based on the user's preferred tags.

Upcoming Features

Playlist support

Technical Details

The project is implemented using MEAN stack, which consists of MongoDB (NoSQL database), Express.js (web application framework running on Node.js), Angular.js (JavaScript MVC framework) and Node.js (the execution environment).
